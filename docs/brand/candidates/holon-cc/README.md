# Holon "Two C's" Logo — Candidate Set

**Selected: `candidate-02-open-mirror.svg`** (2026-05-30). Canonical mark at `docs/brand/holon-logo-mark.svg`; wordmark at `docs/brand/holon-logo-wordmark.svg`; icon set regenerated under `apps/web/src-tauri/icons/`.

The Holon desk product is the cross-team interconnect layer (Core 2 in `holon-engineering`). The mark expresses this literally: **two C's = 互联互通 / interconnect / Hybrid Workforce**. The palette stays sibling to Weizo's gold-crown-on-cream mobile mark — gold `#C69A35` on cream `#F8F6EF`, ink `#1A1A18` reserved for typography and the occasional structural accent. A single jewel accent (`#2E7D52` / `#1F6F9E` / `#7B4FAB`) may appear in duotone variants and nowhere else.

## Candidates

1. **`candidate-01-closed-loop.svg`** — Closed-loop chain: two C's openings facing each other, slightly interlocked. Solid, classic, fits Tauri Windows installer aesthetics.
2. 🟢 **`candidate-02-open-mirror.svg`** — Mirror C's facing outward with a thin ink spine joining the backs. Architectural — "two desks, shared backbone."
3. **`candidate-03-overlapping.svg`** — Duotone overlap: green C behind, gold C front, shared center. Modern and layered; the only jewel-accent candidate.
4. **`candidate-04-figure-eight.svg`** — Braided figure-eight where the C's interweave like an infinity loop. Most dynamic; reads as endless interconnect.
5. **`candidate-05-with-wordmark.svg`** — Header lockup (960×320) using the closed-loop mark plus `Holon` wordmark in Inter 720 and tagline `Local teams. Connected workforce.` Intended for README header / landing page.
