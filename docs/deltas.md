# Deltas — auto-fix queue + global pattern queue

Append-only by the Test/QA loop. Consumed by the Dev loop (local section) + Req loop (global section).

## Conventions

- **Local delta** — a specific defect tied to a specific surface / file / commit. Fixable by a small focused dev agent without re-planning. Format: `- [ ] L-<NNN> · <SHA-short> · <surface>: <one-line issue> → fix: <verb + target>`
- **Global delta** — a pattern across surfaces, architectural drift, or signal that the current iter plan is mis-targeted. Needs Req loop to digest. Format: `- [ ] G-<NNN> · <evidence>: <pattern> → action: <re-plan / new ADR / new debt entry>`
- Mark `[~]` when in flight, `[x] <SHA>` when shipped (for L items) or `[x] <iter-plan-update SHA>` (for G items).

Don't delete shipped lines for 7 days — useful for the Req loop to see what's been moving.
