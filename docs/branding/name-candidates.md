# Manage Your CLI — Western Brand Name Candidates

**Product**: A thin shell that turns the user's CLI AI subscriptions (Claude Code, Codex, Gemini, Qwen, etc.) into a managed team of AI agents. A Secretary the user chats with, plus dynamic employee agents it creates and dispatches. Core concept: **intermediary / acts-on-your-behalf / commands a team**.

**China-market name**: 微作/Weizo (separate — out of scope here).

**IMPORTANT DISCLAIMER**: All trademark and domain checks below are **initial open-web screens only**, NOT legal clearance. They reflect search results as of May 2026. Before committing to any name, commission a proper trademark search in all relevant classes (IC 9, IC 42, IC 35) via a qualified IP attorney.

**Explicitly rejected before evaluation**: *WhatsAI / What'sAI* — high conflict with Meta's WhatsApp; reads as knockoff.

## Recommendation: **Delegra**

### Why Delegra #1

1. **Brandability**: Short (7 characters), pronounceable across English/European markets, no awkward spelling. Passes the "say it over a phone call" test.
2. **Semantic fit**: The root *delegate* is the exact action the product performs. Users intuitively associate delegation with an intermediary who acts on your behalf.
3. **Trademark space**: Initial screen shows no AI/software product conflict. The only name collisions are a personal social handle and an unrelated Russian company.
4. **Domain availability**: Both `delegra.ai` and `delegra.com` appear unregistered / no active indexed site. `.ai` is the preferred TLD for AI tools and appears clear. Registrar verification needed before commit.
5. **Clean coinage**: The word does not exist in common dictionaries, which is good — it can be owned. Compare to "Envoy," "Steward," "Conductor" which are generic English words already saturated.

### Runner-up: **Delegant**

If "Delegra" doesn't clear on registrar check, Delegant is the backup. The phonetic resemblance to "elegant" adds a subtle quality signal. Slightly longer (8 chars) but still clean.

### Cautions before committing

- Do a full USPTO TESS search in IC 9, IC 42, IC 35 for all candidates.
- Run registrar checks (Namecheap/Cloudflare) on `delegra.ai`, `delegra.com`, `delegant.ai`, `delegant.com` — these screens are web-search-based, not registrar API confirmations.
- Check EU IPO and WIPO for international coverage if EU launch is planned.
- Verify no social handle conflicts on X, LinkedIn, GitHub for chosen name.
