# Bug inbox

File bugs here (or just tell Claude in chat — either way they get tracked + fixed).

**To file:** drop a markdown file named `YYYY-MM-DD-<short-slug>.md` in this folder with:

```
# <one-line title>

**Where:** which page / feature
**Saw:** what happened
**Expected:** what you wanted
**(optional)** screenshot / steps
```

Claude scans this folder, turns each into a tracked task, fixes it, and notes the fix
back in the file. Fixed bugs move to `bugs/fixed/`.
